# private-chat
Just a private chat thing that u can use to talk with your friends in the same network

# how to use/tutorial
1. the server and the clients need to have python installed in there system
2. run the server.py on a computer
3. and run the client.py they are the users
4. enter the IP addres of the server (in cmd type ipconfig and copy the "IPv4 Addres..... : "the ip addres" " copy that IP addres and paste it to the input)
5. enter your username
6. and talk with your friends if you even have some
